# SecMobileLab4

Analyse statique d'un APK avec JADX GUI + dex2jar + JD-GUI

Tools used : 

JADX GUI : est actuellement l'outil de référence pour la décompilation et l'analyse statique d'applications Android. 

dex2jar : est un outil utilitaire gratuit et open source indispensable pour le reverse engineering (rétro-ingénierie) d'applications Android.

JD-GUI :  (Java Decompiler Graphical User Interface) est un utilitaire gratuit et open source qui permet de transformer des fichiers compilés Java (fichiers .class) en code source lisible par l'humain.

Android studio :  est l'environnement de développement intégré (IDE) officiel pour la création d'applications sur la plateforme Android
